# 开发环境
1.JDK 1.8.0_281;  
2.Intellij IDEA 2020.3.2;  
3.MySQL 8.0.11;  
4.Maven 3.6.0 +;  
5.Node 10 +;  
6.Redis 3.0.504;  
# 安装
1.`git clone https://github.com/znucleus/ZONE.git`;  
2.配置`application.properties`,其它环境配置仅供开发者使用;  
3.额外配置(settings目录),仅供开发者使用.
# 启动
1.Intellij IDEA Run Application.class 或 Maven .

5.Known RequireJs;
