Z.核校园API 文档

声明：本系统提供的所有公共API，仅供在本系统注册的所有用户使用，使用范围，包括毕业设计及个人项目等非商业性项目。此外，Z.核校园是一个开源项目，当您使用我们提供的服务时，请注明来源，并遵守开源协议。

二、授权API（需登录）

1. 用户基本信息：

GET https://www.zbeboy.top/api/platform/users (域名 www.zbeboy.top 以您当前访问系统为准)

出参：

# 参数名 类型 描述
1 realName string 姓名
2 username string 账号
3 usersTypeId int 用户类型（1：系统；2：学生；3：教师；4：临时用户）
4 verifyMailbox int 是否已验证邮箱（1：已验证；0：未验证）
5 enabled int 是否已注销（1：未注销；0：已注销）
6 accountNonLocked int 是否已锁（1：未锁；0：已锁）
7 avatar string 用户头像路径

2. 学生信息：

GET https://www.zbeboy.top/api/data/student (域名 www.zbeboy.top 以您当前访问系统为准)

出参：

# 参数名 类型 描述
1 studentId int 学生id
2 studentNumber string 学号
3 schoolId int 学校id
4 schoolName string 学校名
5 collegeId int 院id
6 collegeName string 院名
7 collegeAddress string 院地址
8 collegeCoordinate string 院坐标
9 collegeZipCode string 院邮编
10 departmentId int 系id
11 departmentName string 系名
12 scienceId int 专业id
13 scienceName string 专业名
14 gradeId int 年级id
15 grade int 年级
16 organizeId int 班级id
17 organizeName string 班级名

3. 教师信息：

4. 教务校历列表：

GET https://www.zbeboy.top/api/educational/calendar/query-with-college-id/{collegeId} (域名 www.zbeboy.top 以您当前访问系统为准)

出参：

# 参数名 类型 描述
1 state boolean 是否获取成功
2 msg string 失败信息
3 listResult list 数据集
3.1 calendarId string 校历id
3.2 title string 标题
3.3 schoolYear string 学年
3.4 semester byte 学期
3.5 startDate date 开学日期
3.6 endDate date 结课日期
3.7 holidayStartDate date 放假日期
3.8 holidayEndDate date 结假日期
3.9 collegeId int 院id
3.10 username string 发布账号
3.11 publisher string 发布人
3.12 releaseTime string 发布时间
3.13 remark string 备注

5. 教务校历详情：

GET https://www.zbeboy.top/api/educational/calendar/query/{calendarId} (域名 www.zbeboy.top 以您当前访问系统为准)

出参：

# 参数名 类型 描述
1 state boolean 是否获取成功
2 msg string 失败信息
3 listResult list 数据集
3.1 calendarId string 校历id
3.2 title string 标题
3.3 schoolYear string 学年
3.4 semester byte 学期
3.5 startDate date 开学日期
3.6 endDate date 结课日期
3.7 holidayStartDate date 放假日期
3.8 holidayEndDate date 结假日期
3.9 collegeId int 院id
3.10 username string 发布账号
3.11 publisher string 发布人
3.12 releaseTime string 发布时间
3.13 remark string 备注
3.14 releaseTimeStr string 标准格式(yyyy-MM-dd hh:mm:ss) 发布时间
3.15 schoolName string 学校名
3.16 collegeName string 院名
3.17 nowDate date 当前日期
3.18 openWeeks int 上课总共周数
3.19 holidayWeeks int 放假总共周数
3.20 weeks int 当前第几周
3.21 week int 当前星期
